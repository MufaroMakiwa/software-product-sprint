PLAN-IT •
January 2021 This is a web application that I built with two of my friends for a web development class. After being taught the basics of HTML, CSS, and Javascript as well React.js and Node.js, we took part in a class competition under the theme “tag yourself”. PLAN_IT is a progress tracker targeting high school and college students which helps them keep track of tasks that they intend to do on a daily, weekly or monthly basis. To encourage users to use our platform more, we added a feature whereby each user can connect with friends and challenge each other to do tasks, earning points in returns.
